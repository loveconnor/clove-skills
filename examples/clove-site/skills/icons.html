Make every icon earn its meaning.

Use this Agent Skill to design and ship icons that communicate the intended concept, belong to a coherent system, work at real sizes, and remain maintainable.

A polished vector still fails when people misinterpret it.

Define meaning before geometry.

This skill keeps semantics, optical quality, SVG markup, accessibility, security, performance, and rights visible as separate requirements.

When to use this skill

You are drawing, refining, choosing, cleaning, implementing, optimizing, reviewing, or auditing interface icons and SVG systems.

How this skill works

  1. Define the intended concept, context, labels, sizes, states, family, and rights.
  2. Choose library reuse, licensed adaptation, or justified custom design.
  3. Build a coherent grid, keyline, stroke, corner, and optical-sizing system.
  4. Produce intentional, accessible, secure SVG markup and integration.
  5. Verify comprehension, rendering, interaction, packages, and licensing.

Ship cleanly

  • Use a correct viewBox and preserve necessary IDs and semantics.
  • Put actions in native buttons or links with clear accessible names.
  • Keep icon size separate from control and target size.
  • Sanitize untrusted SVG and verify licenses, trademarks, and attribution.
