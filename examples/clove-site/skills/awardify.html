Awardify Agent Skill Make the site unmistakably itself.

Use this Agent Skill to transform an existing website through a site-specific idea, coherent art direction, one primary expressive lever, and deep responsive finish.

Treat award-winning as a quality direction, never a promised result.

Make every detail prove the idea.

This skill protects clarity, agency, access, speed, truth, and maintainability while concentrating expression into a memorable system.

When to use this skill

You are asked to make an existing site more memorable, art-directed, creatively ambitious, award-ready, or less generic through substantial redesign or polish.

How this skill works

  1. Establish the verified goals, content, constraints, and current experience.
  2. Find distinctive material in the existing site and develop structurally different concepts.
  3. Select a direction with evidence and preserve the site’s stable semantic structure.
  4. Define the system, test its riskiest assumption, and implement it in reversible layers.
  5. Polish deeply, remove excess, and verify each quality independently.

Quality gates

  • No fabricated proof, fake controls, or copied creative signature.
  • A usable core across mobile, keyboard, reduced motion, and fallbacks.
  • Representative performance, accessibility, and responsive evidence.
  • Media rights, implementation ownership, and maintenance cost are clear.
