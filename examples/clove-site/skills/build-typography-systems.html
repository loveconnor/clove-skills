Product & interface skill

Set type for real reading.

Choose and implement readable, expressive, accessible typography systems that hold up across content, languages, and devices.

Typography is a relationship between typeface, setting, content, layout, and context.

Start with the body text, then build hierarchy.

Readable type is not a font decision alone. It needs roles, real content, resilient fallbacks, and testing.

Use it when

You are choosing fonts, defining type roles, implementing responsive text, setting form labels, handling data, or reviewing readability and hierarchy.

Working method

  1. Inventory real headings, body copy, labels, data, errors, and languages.
  2. Define reading task, viewport, access needs, and performance constraints.
  3. Choose the body text family, size, measure, and leading together.
  4. Add only the hierarchy roles the content needs.
  5. Test zoom, overrides, loading failure, localization, and sustained reading.

Starting points to test

  • General body text often begins around 1rem–1.125rem.
  • Sustained Latin-script reading often begins around 45–75ch.
  • Body leading often begins around 1.45–1.65.
  • Use contrast, reflow, and text-spacing overrides as non-negotiable checks.
