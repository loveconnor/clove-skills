UX Agent Skill Design for the person, not the screen.

Use this Agent Skill to design flows and interactions around user goals, comprehension, accessibility, recovery, and trust.

Put the right information and action in front of people at the moment they need it.

Make the next step obvious and fair.

Clean UX removes avoidable mental work while keeping context, agency, and recovery visible.

When to use this skill

You are changing a flow, navigation, form, onboarding, setting, dashboard, task model, feedback loop, or accessibility behavior.

How this skill works

  1. Frame the user, goal, context, abilities, and success condition.
  2. Model triggers, decisions, objects, errors, and recovery.
  3. Build hierarchy and structure before visual polish.
  4. Make actions predictable and reduce recall demands.
  5. Test comprehension, task success, confidence, and inclusion.

Baseline

  • Use semantic structure and native controls.
  • Keep focus visible and core tasks usable by keyboard.
  • Show status and consequences in time to act.
  • Do not use false urgency, obstruction, or deceptive defaults.
