Anti-Slop Agent Skill Avoid AI design slop.

Use this Agent Skill to identify and remove generic, fabricated, incomplete, or inaccessible AI-assisted product work while keeping judgment and verification in the process.

Specific, truthful, coherent, complete, tested, and owned.

Build work that holds up after the screenshot.

Use it to establish what is real before turning a prompt into product output.

When to use this skill

You are creating or reviewing a product surface, interface, generated copy, imagery, motion, or code that might look polished but lack evidence, behavior, or ownership.

How this skill works

  1. Establish the user, task, evidence, constraints, and unknowns.
  2. Define the observable outcome and model behavior before polish.
  3. Apply a visual thesis with one purposeful expressive lever.
  4. Stress real states, accessibility, claims, and implementation.
  5. Remove placeholders, fake proof, and residue.

Reject

  • Generic SaaS composition without a product-specific reason.
  • Invented data, testimonials, permissions, or success states.
  • Decoration without hierarchy or a clear job.
  • Happy-path-only work that cannot be explained or maintained.
