SEO Agent Skill Make useful work findable.

Use this Agent Skill to improve public-web discovery and answer-engine visibility by putting the audience’s task, technical eligibility, and the site’s truth first.

Establish eligibility, then provide original value, clear answers, and traceable evidence.

Discovery is a user outcome, not a bag of tricks.

Search and answer engines need clear information, technical consistency, and claims that are safe to understand and quote.

When to use this skill

You are creating public content, mapping search intent, improving metadata or information architecture, adding structured data, or auditing discovery and rendering.

How this skill works

  1. Connect discovery to a real audience need and desired action.
  2. Research the language and intent people actually use.
  3. Verify crawl, rendering, indexing, and mobile eligibility.
  4. Create original information worth retrieving.
  5. Align human-visible and machine-readable facts, then measure outcomes.

Foundations

  • Use accurate titles, headings, stable URLs, and descriptive links.
  • Keep canonical, indexable success URLs consistent.
  • Make essential content and metadata reliably available in HTML.
  • Do not treat schema, crawler hits, or one AI query as proof of success.
