Craft Agent Skill Make it feel like itself.

Use this Agent Skill to turn visual hierarchy, composition, spacing, interaction, and polish into a coherent, recognizable experience.

Create a fitting experience with low avoidable friction and enough character to be recognized.

One clear visual idea, carefully carried through.

Visual quality follows purpose, structure, hierarchy, and deliberate constraints—not decoration alone.

When to use this skill

You are setting visual direction, composing a page, polishing an interface, assessing interaction quality, or diagnosing why a design feels generic or off.

How this skill works

  1. Define the intended experience and its anti-goals.
  2. Establish information order before styling.
  3. Build hierarchy in grayscale.
  4. Constrain type, spacing, color, containers, and motion.
  5. Test real behavior and refine large decisions before small details.

Quality checks

  • Use whitespace to clarify, not merely to empty.
  • Let motion explain cause, continuity, or feedback.
  • Measure comprehension and task performance separately from preference.
  • Check the design in grayscale, at zoom, and with real content.
