How to use Clove.

Clove provides working guidance, not an automatic guarantee of quality. Choose the skills that match the task, follow their complete instructions, and verify the result in context.

  1. 01

    Choose skills that match the task

    Start with the smallest set that covers the work. Add another skill only when the task crosses disciplines.

  2. 02

    Read the complete instructions

    Each skill defines its scope, workflow, research, and non-negotiable standards.

  3. 03

    Resolve important unknowns

    Inspect the available context and ask before making a choice that changes the intended outcome.

  4. 04

    Verify the result

    Test the behavior, content, accessibility, and implementation quality in proportion to the risk.
